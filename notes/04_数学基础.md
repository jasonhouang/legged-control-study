# 04. 数学基础

## 1. 线性代数

### 1.1 矩阵运算

**矩阵乘法**:
```
C = A * B
C[i,j] = Σ_k A[i,k] * B[k,j]

维度: A(m×n) * B(n×p) = C(m×p)
```

**转置**:
```
A^T[j,i] = A[i,j]
```

**逆矩阵**:
```
A * A^-1 = I
A^-1 * A = I

条件: A 必须是方阵且可逆 (det(A) ≠ 0)
```

**伪逆** (Moore-Penrose):
```
A^+ = V * Σ^+ * U^T  (SVD 分解)

其中:
  A = U * Σ * V^T
  Σ^+[i,i] = 1/Σ[i,i]  if Σ[i,i] > ε, else 0
```

**应用**:
- WBC 中的零空间投影: `Z = I - A^+ * A`
- 最小二乘解: `x = A^+ * b`

### 1.2 特征值与特征向量

**定义**:
```
A * v = λ * v

其中:
  λ: 特征值
  v: 特征向量
```

**计算**:
```
det(A - λI) = 0  (特征方程)
```

**性质**:
- 对称矩阵的特征值为实数
- 正定矩阵的特征值 > 0
- 协方差矩阵是半正定的

**应用**:
- 协方差矩阵分析
- 稳定性分析
- 主成分分析 (PCA)

### 1.3 SVD 分解

**定义**:
```
A = U * Σ * V^T

其中:
  U: 左奇异向量 (m×m)
  Σ: 奇异值矩阵 (m×n)
  V: 右奇异向量 (n×n)
```

**性质**:
- Σ 的对角元素是奇异值 (非负)
- U, V 是正交矩阵

**应用**:
- 伪逆计算
- 最小二乘问题
- 降维

### 1.4 四元数

**定义**:
```
q = w + xi + yj + zk = [w, x, y, z]

其中:
  w: 实部
  x, y, z: 虚部
  i² = j² = k² = ijk = -1
```

**单位四元数**:
```
||q|| = √(w² + x² + y² + z²) = 1
```

**旋转表示**:
```
q = [cos(θ/2), sin(θ/2) * n]

其中:
  θ: 旋转角度
  n: 旋转轴 (单位向量)
```

**四元数乘法**:
```
q1 * q2 = [w1*w2 - v1·v2, w1*v2 + w2*v1 + v1×v2]

其中:
  v1 = [x1, y1, z1]
  v2 = [x2, y2, z2]
```

**旋转向量**:
```
p' = q * p * q^-1

其中:
  p = [0, p_x, p_y, p_z]  (纯四元数)
  q^-1 = [w, -x, -y, -z]  (共轭)
```

**与旋转矩阵转换**:
```
R = [1-2(y²+z²),  2(xy-wz),    2(xz+wy)  ]
    [2(xy+wz),    1-2(x²+z²),  2(yz-wx)  ]
    [2(xz-wy),    2(yz+wx),    1-2(x²+y²)]
```

**与欧拉角转换**:
```
Roll (φ):  atan2(2(w*x + y*z), 1 - 2(x² + y²))
Pitch (θ): asin(2(w*y - z*x))
Yaw (ψ):   atan2(2(w*z + x*y), 1 - 2(y² + z²))
```

**应用**:
- 基座姿态表示 (NMPC, WBC)
- IMU 姿态积分
- 插值 (SLERP)

## 2. 刚体动力学

### 2.1 广义坐标

**定义**:
```
q = [q_b, q_j]

其中:
  q_b: 基座位姿 (7维: 位置 + 四元数)
  q_j: 关节角度 (12维)
```

**广义速度**:
```
q̇ = [v_b, ω_b, q̇_j]

其中:
  v_b: 基座线速度 (3维)
  ω_b: 基座角速度 (3维)
  q̇_j: 关节速度 (12维)
```

**广义加速度**:
```
q̈ = [a_b, α_b, q̈_j]

其中:
  a_b: 基座线加速度 (3维)
  α_b: 基座角加速度 (3维)
  q̈_j: 关节加速度 (12维)
```

### 2.2 动力学方程

**拉格朗日方程**:
```
M(q) * q̈ + C(q, q̇) * q̇ + g(q) = τ + J^T * f_ext

其中:
  M(q): 惯量矩阵 (n×n)
  C(q, q̇): 科氏力矩阵 (n×n)
  g(q): 重力向量 (n×1)
  τ: 关节扭矩 (n×1)
  J: 雅可比矩阵 (m×n)
  f_ext: 外力 (m×1)
```

**惯量矩阵 M(q)**:
```
M(q) = Σ_i [m_i * J_v_i^T * J_v_i + I_i * J_ω_i^T * J_ω_i]

其中:
  m_i: 连杆 i 的质量
  I_i: 连杆 i 的惯量张量
  J_v_i: 连杆 i 质心的线速度雅可比
  J_ω_i: 连杆 i 的角速度雅可比
```

**性质**:
- 对称正定
- 稀疏 (大部分元素为 0)

**科氏力矩阵 C(q, q̇)**:
```
C[i,j] = Σ_k c[i,j,k] * q̇_k

其中:
  c[i,j,k] = 0.5 * (∂M[i,j]/∂q_k + ∂M[i,k]/∂q_j - ∂M[j,k]/∂q_i)
```

**性质**:
- 反对称: `q̇^T * (Ṁ - 2C) * q̇ = 0`
- 用于能量守恒分析

**重力向量 g(q)**:
```
g(q) = ∂U/∂q

其中:
  U = Σ_i m_i * g * h_i(q)  (势能)
  h_i(q): 连杆 i 质心的高度
```

### 2.3 雅可比矩阵

**定义**:
```
J = ∂p/∂q

其中:
  p: 末端位置 (3维)
  q: 广义坐标 (n维)
```

**几何雅可比**:
```
J = [J_v, J_ω]

其中:
  J_v: 线速度雅可比 (3×n)
  J_ω: 角速度雅可比 (3×n)
```

**计算**:
```
对于旋转关节 i:
  J_v_i = z_i × (p_end - p_i)
  J_ω_i = z_i

其中:
  z_i: 关节 i 的旋转轴
  p_i: 关节 i 的位置
  p_end: 末端位置
```

**性质**:
- 奇异点: det(J * J^T) = 0
- 零空间: N(J) = {x | J * x = 0}

**应用**:
- 速度映射: `v_end = J * q̇`
- 力映射: `τ = J^T * f_end`
- WBC 中的接触约束

### 2.4 质心动力学

**质心位置**:
```
p_com = (1/m) * Σ_i m_i * p_i

其中:
  m = Σ_i m_i  (总质量)
  p_i: 连杆 i 质心位置
```

**质心速度**:
```
v_com = (1/m) * Σ_i m_i * v_i
```

**质心动量**:
```
h_com = [m * v_com, L_com]

其中:
  L_com = Σ_i [I_i * ω_i + m_i * (p_i - p_com) × v_i]  (角动量)
```

**质心动力学**:
```
ḣ_com = A * h_com + B * f_c + g

其中:
  A = [0, 0; 0, 0]
  B = [I_3, 0; 0, I_3]
  g = [0, 0, -m*g, 0, 0, 0]^T
  f_c: 接触力
```

**优势**:
- 降维: 从 19维 到 6维
- 简化: 不考虑关节细节
- 实时: 求解速度快

## 3. 优化理论

### 3.1 二次规划 (QP)

**标准形式**:
```
min  0.5 * x^T * H * x + f^T * x
s.t. A * x = b          (等式约束)
     C * x ≤ d          (不等式约束)
     lb ≤ x ≤ ub        (边界约束)

其中:
  H: Hessian 矩阵 (n×n, 半正定)
  f: 线性项 (n×1)
  A: 等式约束矩阵 (m×n)
  C: 不等式约束矩阵 (p×n)
```

**求解方法**:
1. **内点法** (Interior Point Method)
   - 多项式时间复杂度
   - 适合大规模问题
   
2. **主动集法** (Active Set Method)
   - 适合小规模问题
   - 热启动快

3. **HPIPM** (High-Performance Interior Point Method)
   - 专为嵌入式优化设计
   - 实时性好

**应用**:
- WBC 中的任务优化
- NMPC 的 QP 子问题

### 3.2 序列二次规划 (SQP)

**思想**: 将非线性规划 (NLP) 转化为一系列 QP

**NLP 问题**:
```
min  f(x)
s.t. g(x) = 0
     h(x) ≤ 0
```

**SQP 迭代**:
```
在第 k 次迭代:
  1. 线性化约束:
     g(x_k + Δx) ≈ g(x_k) + ∇g(x_k) * Δx = 0
     h(x_k + Δx) ≈ h(x_k) + ∇h(x_k) * Δx ≤ 0
  
  2. 二次近似目标:
     f(x_k + Δx) ≈ f(x_k) + ∇f(x_k) * Δx + 0.5 * Δx^T * H_k * Δx
  
  3. 求解 QP:
     min  0.5 * Δx^T * H_k * Δx + ∇f(x_k) * Δx
     s.t. ∇g(x_k) * Δx = -g(x_k)
          ∇h(x_k) * Δx ≤ -h(x_k)
  
  4. 更新:
     x_{k+1} = x_k + α * Δx  (α 为步长)
```

**Hessian 近似**:
```
BFGS 更新:
  H_{k+1} = H_k + (y * y^T) / (y^T * s) - (H_k * s * s^T * H_k) / (s^T * H_k * s)
  
其中:
  s = x_{k+1} - x_k
  y = ∇f(x_{k+1}) - ∇f(x_k)
```

**应用**:
- NMPC 求解器 (OCS2)

### 3.3 分层二次规划 (HQP)

**问题形式**:
```
优先级 1: min ||A_1 * x - b_1||²
          s.t. A_1 * x = b_1

优先级 2: min ||A_2 * x - b_2||²
          s.t. A_1 * x = b_1

优先级 k: min ||A_k * x - b_k||²
          s.t. A_i * x = b_i, ∀i < k
```

**求解方法**:
```
对于优先级 k:
  1. 计算零空间投影:
     Z_{k-1} = I - (A_{1:k-1})^+ * A_{1:k-1}
  
  2. 投影当前任务:
     A_k_proj = A_k * Z_{k-1}
     b_k_proj = b_k - A_k * x_{k-1}
  
  3. 求解投影后的 QP:
     x_k_null = (A_k_proj)^+ * b_k_proj
  
  4. 组合解:
     x_k = x_{k-1} + Z_{k-1} * x_k_null
```

**应用**:
- WBC 中的任务优先级

### 3.4 摩擦锥约束

**定义**:
```
√(f_x² + f_y²) ≤ μ * f_z
f_z ≥ 0

其中:
  f_x, f_y: 切向力
  f_z: 法向力
  μ: 摩擦系数
```

**线性化近似** (金字塔形):
```
f_x ≤ μ * f_z
-f_x ≤ μ * f_z
f_y ≤ μ * f_z
-f_y ≤ μ * f_z
f_z ≥ 0
```

**矩阵形式**:
```
C * f ≤ d

其中:
  C = [1,  0, -μ]
      [-1, 0, -μ]
      [0,  1, -μ]
      [0, -1, -μ]
      [0,  0, -1]
  
  d = [0, 0, 0, 0, 0]^T
```

**应用**:
- NMPC 中的接触约束
- WBC 中的摩擦锥任务

## 4. 数值方法

### 4.1 自动微分

**思想**: 自动计算导数，无数值误差

**前向模式**:
```
对于 f(x):
  x_dot = 1  (种子)
  f_dot = f'(x) * x_dot  (自动计算)
```

**反向模式**:
```
对于 f(x):
  f_bar = 1  (种子)
  x_bar = f'(x)^T * f_bar  (自动计算)
```

**应用**:
- OCS2 中的雅可比计算
- NMPC 中的梯度计算

### 4.2 积分方法

**欧拉法**:
```
x_{k+1} = x_k + f(x_k, t_k) * dt

精度: O(dt)
```

**龙格-库塔法** (RK4):
```
k1 = f(x_k, t_k)
k2 = f(x_k + 0.5*dt*k1, t_k + 0.5*dt)
k3 = f(x_k + 0.5*dt*k2, t_k + 0.5*dt)
k4 = f(x_k + dt*k3, t_k + dt)

x_{k+1} = x_k + (dt/6) * (k1 + 2*k2 + 2*k3 + k4)

精度: O(dt^4)
```

**应用**:
- IMU 积分 (状态估计)
- 动力学仿真

### 4.3 插值方法

**线性插值**:
```
y = y_0 + (y_1 - y_0) * t / (t_1 - t_0)
```

**三次样条插值**:
```
S(t) = a + b*t + c*t² + d*t³

条件:
  S(t_i) = y_i
  S'(t_i) = S'_{i+1}(t_i)  (一阶导数连续)
  S''(t_i) = S''_{i+1}(t_i)  (二阶导数连续)
```

**应用**:
- 轨迹规划
- 摆动腿轨迹
